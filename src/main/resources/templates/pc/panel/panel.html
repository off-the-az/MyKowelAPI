Other requests


User:


  • POST Request - /user (Need all about users)
  • PUT Request - /user/{token}
  • DELETE Request - /user/{token}

  • POST Request - /user/login (Just login and password)
  • POST Request - /user/logout (Just login and password)

Market:


  • POST Request - /market/add (Need all about item of market)
  • PUT Request - /market/update/{id} (All about item market with ID)
  • DELETE Request - /market/delete/{id} (Need only ID)

Events:


  • POST Request - /events/add (Need all about event)
  • DELETE Request - /events/delete/{id} (Need only ID)

  • In progress:
  • PUT Request - /events/update/{id} (Need all about event with ID)

Contacts:


  • POST Request - /contacts/add (Need all about contact)
  • PUT Request - /contacts/{id} (Need all about contact with ID)
  • DELETE Request - /contacts/delete/{id} (Need only ID)



To test not from live branch - just add before name of table(ex. - /users) /dev element of dev branch
